Tutorials

Batch basics

Batch (.bat) files are plain text files executed by Windows CMD. Common commands: echo, set, if, for, goto, call.

Variables & input

@echo off
```

set /p name=Enter your name:
echo Hello %name%
pause
```

Control flow

@echo off
```

set /p c=Choose 1 or 2:
if "%c%"=="1" goto one
if "%c%"=="2" goto two
goto end
:one
echo You chose 1
goto end
:two
echo You chose 2
:end
pause
```

.BAT Reference

Echo

echo prints text. Use @echo off at top to quiet the command display.

Set & variables

Assign: set name=value. Read: %name%. Use delayed expansion for runtime changes (!var!).

If / For / Goto

Conditionals: if, loops: for /f, labels: :label and jump with goto label.

Tips

  • Quote paths that contain spaces.
  • Test on a VM before running scripts that modify files or system settings.
  • Prefer PowerShell for advanced scripting and better safety/features.

Examples

Create backup of folder

@echo off
```

set src=C:\MyProject
set dst=C:\Backup\MyProject_%date:~10,4%-%date:~4,2%-%date:~7,2%
xcopy "%src%" "%dst%" /E /I /Y
echo Backup created: %dst%
pause
```

Safe IP generator (private ranges)

@echo off
```

rem Generates a PRIVATE LAN IP for testing
set /a R=%random%
set /a choice=R %% 3
if %choice%==0 (
set /a b=%random% %% 256
set /a c=%random% %% 256
set /a d=%random% %% 256
echo 10.%b%.%c%.%d%
) else if %choice%==1 (
set /a b=16 + (%random% %% 16)
set /a c=%random% %% 256
set /a d=%random% %% 256
echo 172.%b%.%c%.%d%
) else (
set /a c=%random% %% 256
set /a d=%random% %% 256
echo 192.168.%c%.%d%
)
pause
```
